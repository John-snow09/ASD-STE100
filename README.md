# ASD-STE100 Checker & AI Rewriter

An advanced NLP tool designed to check technical manuals for compliance with the **ASD-STE100** (Simplified Technical English) standard. It identifies non-compliant terminology and uses **Llama 3** to provide context-aware rewrites.

## 🚀 Features
- **Vector-Based Table Filtering:** Automatically ignores data inside tables to focus only on procedural text.
- **18k+ Rule Dictionary:** High-speed scanning using a comprehensive STE dictionary.
- **AI-Powered Rewriting:** Integration with local Llama 3 models for intelligent sentence simplification.
- **PDF Annotation:** Generates a new PDF with red highlights and sticky note corrections.
